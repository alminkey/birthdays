# CoreWire - AI News Portal

## 📋 Project Overview

**CoreWire** je besplatan, nepristrasni AI news agregator koji analizira više izvora i pruža objektivne zaključke.

### Tagline
"Core facts, wired globally"

---

## 🎯 Ciljevi

- Neutralan informativni medij zasnovan na činjenicama
- Analiza više izvora → objektivni zaključci
- Odgovori na: Kako, Kad, Zašto
- Besplatan pristup za sve korisnike

---

## 🏗️ Tehnička Arhitektura

### Frontend (Mobile-Ready)
```
Next.js 14 + React
├── Responsive design
├── PWA support (Progressive Web App)
├── API-first consumption
└── Future: React Native mobilna app
```

### Backend (API-First)
```
FastAPI (Python)
├── RESTful API
├── WebSocket za real-time updates
├── Mobile-ready endpoints
└── Sinhronizacija ready
```

### Database
```
PostgreSQL
├── News articles
├── Sources
├── User preferences
└── Analytics
```

### AI Engine
```
DeepSeek API
├── News aggregation
├── Bias analysis
├── Summary generation
└── Fact extraction
```
