# CoreWire

AI-powered news aggregation and analysis platform.

## Overview

Corewire is an unbiased, fact-focused news platform that uses AI to analyze multiple sources and provide objective insights.

### Features

- AI-powered news aggregation
- Multi-source analysis
- Bias detection
- Mobile-ready interface
- API-first architecture

## Tech Stack

- Frontend: Next.js
- Backend: FastAPI
- Database: PostgreSQL
- AI: DeepSeek

## Project Structure

```
corewire/
├── backend/     # FastAPI server
├── frontend/    # Next.js client
├── docs/        # Documentation
└── scripts/     # Utility scripts
```
